## What is PreVeil?

PreVeil is an end-to-end encrypted email and file sharing platform designed for organizations that handle sensitive data and regulated information[citation:1][citation:5]. It provides FedRAMP Moderate Baseline Equivalent security using FIPS 140-2/140-3 validated encryption modules[citation:1][citation:2]. Unlike traditional solutions requiring complex portals or passwords, PreVeil integrates directly with Outlook and Gmail, allowing users to send and receive encrypted messages using their existing email addresses[citation:7][citation:9].

Infrastructure teams managing secure communications benefit from PreVeil's zero-trust architecture and compliance solutions for CMMC, ITAR, DFARS 7012, HIPAA, and IRS regulations[citation:1][citation:4][citation:8]. System administrators appreciate the low-cost, easy-to-deploy platform that saves up to 75% compared to GCC High[citation:1][citation:2].

## Key Features

| Feature | Description |
|---------|-------------|
| **End-to-End Encryption** | Only you and your recipients can decrypt messages. No one else—not Google, not even PreVeil—can access your data[citation:7][citation:8]. |
| **Outlook and Gmail Integration** | Add PreVeil’s encrypted mailbox directly to your existing Outlook or Gmail client. Send and receive encrypted emails without logging in or changing your address[citation:5][citation:7]. |
| **PreVeil Drive** | Encrypt, store, and share files of any kind, including CAD and BIM. Integrates with Windows Explorer and browsers[citation:1][citation:5]. |
| **Auto-Encrypt** | Add a predefined keyword (CUI, ITAR, FCI, ENCRYPT) to any email subject line and PreVeil automatically encrypts it[citation:5]. |
| **Compliance Accelerator** | Pre-filled, C3PAO-validated documentation (SSP, SOP, POA&M) that maps directly to CMMC and NIST 800-171 controls, cutting documentation work by up to 60%[citation:1][citation:2][citation:5]. |
| **Zero Trust Security** | Designed using NSA-recommended Zero Trust Security to protect emails even when servers, admins, and passwords are compromised[citation:1][citation:2]. |
| **Mobile App** | Access encrypted email and files from any iPhone, iPad, or Android device with no additional cost[citation:5][citation:8]. |
| **Third-Party Sharing** | Send encrypted emails to anyone outside the PreVeil network—including GCC High users—without the recipient needing a PreVeil account[citation:5][citation:8]. |
| **Revoke Access** | Revoke access to sent emails and files at any time. Set expiration dates for messages[citation:5]. |

## Installation Guide

### Prerequisites

- Windows 10 or above (Windows 8 and above supported)[citation:9][citation:12]
- Administrator permissions required for installation[citation:12]
- Web browser: Google Chrome, Microsoft Edge, or Mozilla Firefox[citation:9][citation:12]

### Step-by-Step Installation

**Step 1:** Go to `https://www.preveil.com/download`[citation:9][citation:12].

**Step 2:** Check the box to agree to the Terms of Use and Privacy Policy, then click the **Download for Windows** button[citation:9][citation:12].

**Step 3:** Locate the downloaded file in your Downloads folder and double-click it to run the installer[citation:9][citation:12].

**Step 4:** When the confirmation box appears asking if you want to allow the app to make changes, click **Yes**[citation:9].

**Step 5:** Review the PreVeil License Agreement, select **I accept the agreement**, and click **Next**[citation:9][citation:12].

**Step 6:** Click **Install** to begin the installation[citation:9][citation:12].

**Step 7:** Wait for the installation to complete. This typically takes a few minutes[citation:9][citation:12].

**Step 8:** When the installation is complete, click **Finish**[citation:9][citation:12].

### Create Your PreVeil Account

**Step 1:** Click the PreVeil shortcut on your desktop or open a browser and go to `preveil.com/app`[citation:9][citation:12].

**Step 2:** Click the **Create Account** button[citation:10].

**Step 3:** Enter the email address you want to use as your primary PreVeil ID and click **Send Verification Email**[citation:10].

**Step 4:** Access your email to get the verification code (check Spam/Junk folder if needed)[citation:10].

**Step 5:** Enter the verification code and click **Verify**[citation:10].

**Step 6:** Enter your name and accept the Terms of Service and Privacy Policy, then click **Create Account**[citation:10].

**Step 7:** Create a **Recovery File** to protect your PreVeil encryption key[citation:10].

### Optional: PreVeil for Gmail Browser Extension

**Note:** Requires PreVeil Desktop Application[citation:7].

**Step 1:** Install the PreVeil desktop application first[citation:7].

**Step 2:** Add the PreVeil Gmail browser extension from the Chrome Web Store[citation:7].

**Step 3:** Messages are now automatically encrypted end-to-end within Gmail[citation:7].
